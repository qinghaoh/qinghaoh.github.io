# 概率模型

## 无限牌堆模型

在此理想模型下，我们假设任意一个时刻，牌堆中的卡牌数量无限多，且有均匀分布的4种花色和13种点数。换言之，任何时刻牌堆顶的牌是某种花色的概率为常数\\(1/4\\)，是某种点数的概率为常数\\(1/13\\)。

## 军争篇模型

在实际的游戏中，卡牌数量是有限的。我们在此只考察军争篇的卡牌模型。军争篇卡牌共有160张，具体分布如下（引用自[萌娘百科](https://zh.moegirl.org.cn/zh-hans/%E4%B8%89%E5%9B%BD%E6%9D%80/%E7%89%8C%E5%A0%86%E7%BB%93%E6%9E%84)）：

![军争篇卡牌分布](/assets/sgs/junzheng.png)

在此模型下，为了简化计算，我们排除宝物牌`木牛流马`。在此前提下，卡牌分为4种花色，每种花色40张，且在游戏中的任意一个时刻，我们可以认为牌堆中的牌的花色和点数都是均匀分布的。这种简化的合理性在于：场上的牌分布情况千差万别，我们无法对牌堆做精确计算。因此，我们只取统计意义上的近似结果。也正因如此，本文中所有的小数结果仅保留四位有效数字。

# 武将技能概率

本章中，所有的讨论都是首先针对无限牌堆模型的理想情况，之后再尽量讨论军争篇模型。

## 甄姬 洛神

![甄姬](/assets/sgs/empress_zhen.png)

洛神：回合开始阶段，你可以进行判定：若为♠或♣，立即获得此牌，并可以再次使用洛神——如此反复，直到出现♥或♦为止。

### 理想模型

定义洛神获得的牌数为\\(Y\\)，并定义一次伯努利试验（[Bernoulli trial](https://en.wikipedia.org/wiki/Bernoulli_trial)）的“成功”为判定红牌，而“失败”为判定黑牌。很显然，\\(Y\\)属于“失败”次数，且其概率分布符合几何分布（[Geometric distribution](https://en.wikipedia.org/wiki/Geometric_distribution)）的第二种形式，其中\\(p = 1/2\\)。

概率质量函数（PMF, Probability mass function）:

$$p_{Y}(k) = (1-p)^{k}p = (\frac{1}{2}) ^ {k+1}, \quad k \in \mathbb{N}_0$$

数学期望（Expected value）和方差（Variance）分别为：

$$\operatorname{E}(Y) = {\frac{1-p}{p}} = 1,\qquad \operatorname{var}(Y)={\frac {1-p}{p^{2}}} = 2$$

这说明在理想情况下，洛神所得的牌数平均为1张。

接下来我们考察甄姬的战功“洛神赋”：使用甄姬一回合内发动洛神在不被改变判定牌的情况下连续判定黑色花色至少8次。

$$p_{Y}(k \ge 8) = 1 - \sum_{k = 0}^7{p_Y(k)} = \frac{1}{256} \approxeq 0.003906$$

### 军争模型

定义牌堆中的牌数为\\(n\\)，牌堆中黑色花色的牌数为\\(m\\) (\\(m < n\\))，则PMF为：

$$p_{Y}(k) = \frac{\binom{m}{k}}{\binom{n}{k}} \frac{n-m}{n-k}, \quad k=0,1,\ldots,m$$

数学期望为：

$$\operatorname{E}(Y) = \sum_{k=0}^{m}kp_{Y}(k)
 = \sum_{k=1}^{m}\frac{k\binom{m}{k}}{\binom{n}{k}} \frac{n-m}{n-k} \\
 = (n-m) \sum_{k=1}^{m}\frac{m\binom{m-1}{k-1}}{\binom{n}{k}(n-k)}
 = m(n-m) \sum_{k=1}^{m}\frac{\binom{m-1}{k-1}}{n\binom{n-1}{k}} \\
 = \frac{m(n-m)}{n}\frac{n}{(n-m)(n-m+1)}
 = \frac{m}{n-m+1}$$

如果我们假定\\(m = n/2\\)，那么：

$$\operatorname{E}(Y) = \frac{1}{1+\frac{2}{n}} < 1$$

可见，\\(\operatorname{E}(Y)\\)与\\(n\\)正相关，且小于理想模型下洛神牌的数学期望。

在军争篇模型中，共有160张牌。如果对所有可能的牌堆卡牌数取平均：

$$\overline{\operatorname{E}} = \operatorname{E}(Y) = \frac{1}{160} \sum_{n=1}^{160}\frac{1}{1+\frac{2}{n}} \approxeq 0.9479$$

结论：在实际的军争篇游戏中，甄姬洛神的额外摸牌不稳定，而且其期望要小于周瑜和貂蝉。
