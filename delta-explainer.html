Key rules to remember
ITM — In the Money
Call: S > K → has intrinsic value
Put: S < K → has intrinsic value
Delta closer to 1.0 (Call) or −1.0 (Put)
High exercise probability
ATM — At the Money
S ≈ K → intrinsic value ≈ $0
All value is time value
Delta ≈ ±0.5
~50% exercise probability
OTM — Out of the Money
Call: S < K → intrinsic value = $0
Put: S > K → intrinsic value = $0
Delta closer to 0
Low exercise probability
